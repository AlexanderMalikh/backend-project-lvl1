# Brain Games
**Hexlet Node.js project**

## Installation guide
### Install from NPM

`$npm i hexletbackendproject1`

### Install from GitHub

`$git clone https://github.com/AlexanderMalikh/backend-project-lvl1.git`
`$make publish`
`$npm link`
